# 示例输入：访客预约首页

请使用 `figma-ui-restore`。

## Figma 输入

- 页面类型：后台管理首页 / 工作台
- 画布基准：1440
- 左侧有菜单，右侧是业务 content
- 项目已有后台 Layout，因此本次只生成右侧 content

## 业务模块

1. 欢迎模块
   - 左侧欢迎语和日期
   - 右侧帮助文档入口
   - 始终保持左右布局，不换行

2. 快捷入口卡片
   - 新建邀约
   - 访客自助
   - 大屏横向排列，小屏纵向排列

3. 预警与统计模块
   - 左侧未提交预约单预警
   - 右侧两个统计卡
   - 中屏时统计卡换到第二行并均分宽度

4. 最近访客列表
   - 列表 item 数据驱动
   - 标题、公司、地点长文本需要省略和 Tooltip

## 技术栈

- Vue3
- TypeScript
- Less
- Arco Design
- 已有 Layout
- 已有全局 Table 组件

## 要求

先输出开发前确认报告，不要直接写代码。
